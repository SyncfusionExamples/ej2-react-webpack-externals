# ej2-react-webpack-externals
Demo sample for the EJ2 react webpack externals. To get further information, check out the [getting started](https://ej2.syncfusion.com/react/documentation/getting-started/webpack-externals/) document.

### Step: 1 

Open the command prompt from the required directory, and run the following command to clone the Syncfusion® react (Essential® JS 2) webpack project from

`git clone https://github.com/SyncfusionExamples/ej2-react-webpack-externals.git`

### Step: 2

After cloning the application in the ej2-react-webpack-externals folder, run the following command line to navigate to the ej2-react-webpack-externals folder.

`cd ej2-react-webpack-externals`

## Installation

Use the following command to install the dependent npm packages from the command prompt.

`npm install`

## Run the sample

Run the application by using following command:

`npm start`

## Build

Build the application by using following command:

`npm run build`
